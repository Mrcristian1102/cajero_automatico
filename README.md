# Cajero Automático en Python

Programa sencillo en **Python** que simula un cajero automático desde la terminal.

## Funciones

* Consultar saldo
* Retirar dinero
* Depositar dinero

El programa inicia con un saldo de **$1000** y permite realizar varias operaciones.

## Ejecutar el programa

```bash
python cajero.py
```
