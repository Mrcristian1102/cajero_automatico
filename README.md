# Cajero Automático

## Descripción

Este proyecto es un programa en **Python** que simula un **cajero automático** en la consola.
Permite al usuario realizar operaciones básicas como consultar saldo, retirar dinero o depositar dinero.

## Funcionalidades

El sistema permite al usuario:

* Consultar el saldo disponible
* Retirar dinero del saldo
* Depositar dinero a la cuenta
* Realizar varias operaciones en una misma ejecución

## Requisitos técnicos

* Uso de variables
* Tipos de datos básicos (`int`, `float`)
* Entrada de datos por consola (`input`)
* Salida de información en consola (`print`)
* Uso de estructuras condicionales (`if`, `elif`, `else`)
* Uso de ciclos (`for`)

## Resultado esperado

Al ejecutar el programa el usuario podrá:

* Ver su saldo inicial
* Elegir una opción del menú
* Realizar la operación seleccionada
* Ver el saldo actualizado después de cada operación
